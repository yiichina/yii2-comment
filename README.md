安装
----

安装这个扩展的首选方法是通过 [composer](http://getcomposer.org/download/)。

可以运行

```
composer require --prefer-dist yiichina/yii2-comments "*"
```

也可以添加

```
"yiichina/yii2-comments": "*"
```

到你的 `composer.json` 文件的包含部分。
